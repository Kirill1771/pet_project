Online shop 'Candy shop'

Интернет-магазин на Django

Tools:
Back-End: Python 3.8, Django 4.0.6, Django Rest Framework 3.13.1
Database: MySQL

virtualenv venv
pip install -r requirements.txt
python manage.py runserver

Features:
Authentication system for user registration, login, profile change and password change or reset
Shopping cart system with Django sessions
Forms for placing orders
API category, production, cart
